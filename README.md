但是在serverUser中，用这个方法就跳转不过去。我就只有用一个笨办法先做：既然在登录页面跳转不过去，就只有
提示用户（当然这个只是自己做着玩的，我就是用户哈）注册成功或者失败，然后在html页面中添加一个a标签的登录，
让用户自己去登录，这样显然不太好，目前我想先做完这个小项目，如果之后学习node的express比较深入的话，再过来修正。
